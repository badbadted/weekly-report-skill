週報自動滾動生成 Skill

Claude Code Skill — 貼上上週週報，自動產出本週版本

安裝

1

Clone 專案

git clone https://github.com/badbadted/weekly-report-skill.git
2

在你的工作專案中建立 Skill 目錄

mkdir -p .claude/skills/weekly-report
3

複製 SKILL.md 到 Skill 目錄

cp weekly-report-skill/SKILL.md .claude/skills/weekly-report/SKILL.md
完成後在 Claude Code 中輸入 /weekly-report 即可啟用。

使用流程

啟動 Skill
/weekly-report
➔
貼上上週週報
➔
AI 自動滾動
生成本週初版
➔
回報差異
➔
產出完整週報
你只需要告訴 AI「跟上週不一樣」的部分，其餘全部自動沿用。

差異更新範例

滾動完成後，一次告知所有變動即可：

C1 的第 1 項完成度改 80%，實際工時 6hr
新增 E 會議 8 場 12HR
F 新增：請款單系統 SA02F04 SPEC 修訂 8hr
下週預計：C2 某專案 8hr，排 2 個項目各 4hr

指令一覽

指令說明

/修改局部修改已產出的週報

/格式顯示格式參照範例
/指令列出所有可用指令

自動功能

功能說明
工時驗算每次產出自動加總，子項 ≠ 總計時提醒
待辦管理未完成自動轉入待辦，已排程自動移出
異常偵測週工時 >44hr / <30hr、完成度異常等自動標記
預設沿用A/B 數值、部門事務工時、負責人等自動從上週帶入
